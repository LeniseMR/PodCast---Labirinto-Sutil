# PodCast---Labirinto-Sutil
Desafio de Projeto - Criando um Podcast com IA generativa
Prompt 1 - Você é um roteirista de podcast, e vamos criar um podcast de educação, focado em aprendizagem para toda a vida e eu gostaria de uma ajuda sua para criar 5 sugestões de nomes criativos para um podcast de aprendizagem, psicologia e desenvolvimento pessoal. O podcast vai falar sobre reflexões e insights nas áreas de aprendizagem para a vida toda e escolhas inteligentes para o desenvolvimento pessoal para pessoas entre 30 e 50 anos.

{REGRAS}
- O nome deve ser enxuto, um nome e um subtítulo
- O nome deve conter alguma palavra forte que remeta a escolhas inteligentes e aprendizado

{REGRAS {REGRAS NEGATIVAS}

- Não quero que o título contenha palavras em inglês
- Não quero que utilize nenhuma das palavras nessa lista: aprendizagem e estratégia.

o formato do roteiro deve ser
[INTRODUÇÃO]
[CURIOSIDADE 1]
[CURIOSIDADE 2]
[FINALIZAÇÃO]

{REGRAS}

- no bloco [INTRODUÇÃO] substitua por uma introdução do tema aprendizagem para a vida toda.
- no bloco [CURIOSIDADE 1] substitua por uma curiosidade sobre o tema aprendizagem para a vida toda.
- no bloco [CURIOSIDADE 2] sobre uma ferramenta utilizada para fazer escolhas inteligentes e estratégicas sobre desenvolvimento pessoal.
- no bloco [FINALIZAÇÃO] substitua por uma despedida com o final 'Eu sou a Lenise e esse foi mais um avanço no Labirinto Sutil'
- use termos de fácil explicação
- O podcast vai ser apresentado somente por uma pessoa, chamada Lenise
- O podcast deve ser curto

{REGRAS NEGATIVAS}

- Não use muitos termos técnicos
- Não ultrapasse 5 minutos de duração
